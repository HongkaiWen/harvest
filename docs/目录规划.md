# 目录规划

### 概述

分为源目录和目标目录，应用把照片/视频从源目录搬运到目标目录，在搬运过程中，在目标目录中按照日期/地点等维度对照片进行整理。

### 规划

```
/
/harvest
    /bin
    /libs
    /config
    /logs
    /data

/media-center
    /repository
        /photos
            /yyyy
                /mm
        /videos
            /yyyy
                /mm
    /stage
    /backup
        /yyyy-mm-dd_hh_mm_ss
    /error
        /yyyy-mm-dd_hh_mm_ss
            /error
            /ignore

```
